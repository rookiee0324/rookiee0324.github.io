Learn to establish Blog by GitHub + Hexo

Download

  1. Node.js
  2. Git

    visit GitHub

    check ipaddress
    WIN+X - Windows Powershell(admin)
    notepad - C:Windows/system32/drivers/etc/hosts
    github.com - IP Address 140.82.113.4

Install and check

WIN+R - cmd
node -v / npm -v / git --version

Sign up GitHub

1.Create account ++Username++ / ++Email++ ==on the Web==
2.Right click - Git Bash here ==on the Computer==
3.git config --global user.name "GitHub ++Username++"
4.git config --global user.email "GitHub ++Email++"
5.ssh-keygen -t rsa -C "GitHub ++Email++" Enter…
6.Copy ++C:\User\User\.ssh\id_rsa.pub++ by notepad
7.Sign in to GitHub - ++Settings++ - ++SSH and GPG keys++ - ++New SSH key++
8.Paste the Copy - ++add SSH key++
9.Git Bash $ ssh -T git@github.com to see “Are you sure…” - Enter yes - relpy “Hi Username! You’ve successfully…”

Create GitHub repository

Repository name = username.github.io

Install Hexo

1.Create a new local folder
2.Right click - Git Bash here
3.npm install -g hexo-cli
4.be patient for waiting install
5.hexo init
6.npm install
7.hexo g == hexo generate
8.hexo s == hexo server
9.http://localhost:4000
if not
hexo server -p 5000
http://localhost:5000

Connect Hexo to GitHub

npm install hexo-deployer-git --save
Edit ‘_config.yml’ by Notepad

deploy:
type: git
repository: git@github.com:username/username.github.io.git
branch: master

hexo d == hexo deploy
https://rookiee0324.github.io/ (Subdomain of GitHub)
